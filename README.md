# install-ghalint

An action to install [suzuki-shunsuke/ghalint](https://github.com/suzuki-shunsuke/ghalint).

## Usage

```yaml
  - uses: Lucky3028/install-ghalint@v1
    with:
      version: 0.2.0
```

### Inputs

|name|description|required|
|---|---|:---:|
|version|The version of ghalint to install. (ref. 1.1.0) If not provided, use the latest.|â¬œ|
